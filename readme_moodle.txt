Moodle maintainer: WIRIS Europe (Maths for more S.L)

## Installation instructions

This plugin bundles WIRIS MathType PHP and Javascript libraries separately as defined on the `thirdpartylibs.xml` file on this
project. 

The library is automatically shipped with the filter, so you don't need to do any installation steps.
